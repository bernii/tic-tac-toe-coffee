Tic Tac Toe.coffee

Welcome to the simple game of Tic Tac Toe.
It was created as a little excercise with AI algorithms (Negamax) and dynamic programming. Two of Hakims' cool visual experiments were used for fun stuff in the UI.

If you want the computer to start first use the Computer start button, otherwise just make your move. You can change the game difficutly with the Computer AI setting. Below the board you can see the visualization of the tree that is analyzed by Negamax (when it is selected) that allows the computer to beat you ;-). Have fun!
